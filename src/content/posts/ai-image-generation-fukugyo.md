AI画像生成が進化して、素人でもプロ並みの画像が作れるようになった。これを使って稼げるのか、プラットフォームごとに調べてみた。

## AI画像生成ツールの比較

まず、どのツールを使うべきか整理した。

| ツール | 料金 | 特徴 | 商用利用 |
|--------|------|------|---------|
| Midjourney | 月10ドルから | 高品質。アート向き | 有料プランならOK |
| DALL-E 3 | ChatGPT Plus（月20ドル） | 使いやすい。テキスト描画が得意 | OK |
| Stable Diffusion | 無料（ローカル） | カスタマイズ性が高い | OK（モデルによる） |
| Adobe Firefly | 無料から月680円 | Adobe製品と連携 | OK |
| Canva AI | 無料から月1,000円 | デザイン初心者向き | OK（有料プラン） |

商用利用を考えるなら、ライセンスが明確なMidjourneyかAdobe Fireflyが安全。

## ストックフォトで稼ぐ方法

### 各サービスのAI画像への対応状況

| サービス | AI画像 | AI表記 | 報酬単価 |
|---------|--------|--------|---------|
| Adobe Stock | OK | 「AI生成」ラベル必須 | 1DL 30から50円 |
| Shutterstock | OK | 専用カテゴリあり | 1DL 25から40円 |
| PIXTA | NG | 2024年から受付停止 | - |
| 写真AC | OK | 特に制限なし | 1DL 3から11円 |
| iStock | OK | メタデータに記載必須 | 1DL 20から40円 |

PIXTAがNGなのは大きい。日本のストックフォト市場ではPIXTAのシェアが高いので、日本向けに稼ぎたい場合は不利。

### 実際にどれくらい稼げるか

ストックフォトで稼いでいる人の声を調べた。

| 投稿数 | 月間DL数 | 月収 | 期間 |
|--------|---------|------|------|
| 100枚 | 10から30DL | 300から1,500円 | 1から3ヶ月目 |
| 500枚 | 50から150DL | 1,500から7,500円 | 6ヶ月目 |
| 1,000枚 | 100から300DL | 3,000から15,000円 | 1年目 |
| 3,000枚以上 | 300から1,000DL | 1万から5万円 | 2年目以降 |

大量にアップして、気長に待つビジネスモデル。1枚あたりの単価が低いので、数で勝負する必要がある。

### 売れやすい画像のジャンル

| ジャンル | 需要 | 競合 |
|---------|------|------|
| ビジネスシーン（会議、オフィス） | 非常に高い | 多い |
| テクノロジー（AI、データ） | 高い | 増加中 |
| 自然・風景 | 中程度 | 非常に多い |
| 食べ物 | 高い | 多い |
| 抽象的な背景・テクスチャ | 中程度 | AI向き |
| 季節イベント（クリスマスなど） | 時期限定で高い | 多い |

AI画像が特に強いのは「抽象的な背景」「テクスチャ」「コンセプト画像」。リアルな人物写真はまだAIだと不自然さが残ることが多い。

## LINEスタンプで稼ぐ方法

### 基本情報

| 項目 | 内容 |
|------|------|
| スタンプ価格 | 120円から610円 |
| クリエイターの取り分 | 売上の35% |
| 1セットの枚数 | 8/16/24/32/40枚から選択 |
| 審査期間 | 1から3日 |

### 収益シミュレーション

120円のスタンプが1セット売れた場合:
- 売上: 120円
- クリエイターの取り分（35%）: 42円

月に100セット売れたとして、月収は4,200円。1,000セット売れても42,000円。

### LINEスタンプ市場の現実

- 2015年頃がピーク。現在は市場が飽和している
- 登録クリエイター数は200万人以上
- AI生成スタンプが大量に出回っており、差別化が極めて困難
- 月収1万円以上稼いでいるクリエイターは全体の数%

### AI生成スタンプを作る手順

1. Midjourneyで統一感のあるキャラクターを生成（プロンプトにスタイルの一貫性を指定）
2. 背景を透過処理（Canvaやremove.bgで可能）
3. サイズ調整（370x320px。メインが96x74px）
4. LINEクリエイターズマーケットに登録
5. 審査を通過すれば販売開始

注意点: キャラクターの一貫性を保つのが難しい。同じプロンプトでも毎回微妙に違うキャラが生成される。Stable DiffusionのControlNetなどを使えば改善できるが、学習コストが高い。

## NFTアートとして売る方法

AI画像をNFT（非代替性トークン）として販売する方法もある。

### 現状

- NFT市場は2022年のブームから大幅に縮小
- 2024年以降、取引量は最盛期の5%以下
- 著名アーティスト以外のNFTはほぼ売れない
- AI生成NFTに対する評価は低い

結論: 今からNFTアートで稼ぐのは非現実的。

## まとめ

AI画像生成で稼ぐこと自体は可能だが、どのプラットフォームでも単価が低く、大量生産が前提になる。

| 方法 | 現実的な月収 | 必要な投稿数 | おすすめ度 |
|------|-----------|-----------|----------|
| ストックフォト | 1,000から1万円 | 500枚以上 | まあまあ |
| LINEスタンプ | 0から5,000円 | 数セット | 厳しい |
| NFTアート | ほぼ0円 | - | 非推奨 |

「趣味の延長でお小遣い程度」なら悪くないが、安定した副業収入を期待するのは難しいと感じた。
