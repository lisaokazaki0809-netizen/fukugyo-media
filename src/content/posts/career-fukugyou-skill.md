副業は「お金を稼ぐ」だけでなく「スキルを身につける」手段でもあるはず。実際にキャリアにどう活きるのか調べた。

## 副業の種類別: 身につくスキル

| 副業の種類 | 身につくスキル | 本業への転用先 |
|-----------|-------------|------------|
| ブログ運営 | ライティング、SEO、マーケティング、データ分析 | 広報、マーケ、企画 |
| 動画編集 | クリエイティブ、ツール操作、納期管理 | 広報、PR、社内研修 |
| プログラミング | 技術力、問題解決、論理的思考 | DX推進、業務改善 |
| コンサルティング | 提案力、コミュニケーション、業界知識 | 営業、経営企画 |
| せどり・物販 | 市場分析、在庫管理、数値管理 | 営業、購買、経理 |
| 翻訳・通訳 | 語学力、異文化理解 | 海外事業、グローバル対応 |
| デザイン | デザイン思考、UI/UX | 商品企画、マーケ |

## 副業スキルが本業に活きた具体的な事例

### 事例1: ブログのSEO知識で社内異動

営業職の会社員がブログを副業で始めた。SEOとWebマーケティングの知識を独学で身につけた結果、社内のWebマーケ部門への異動が実現。年収は50万円アップ。

### 事例2: 副業プログラミングでDX推進メンバーに

経理職の会社員がプログラミングを副業で学んだ。ExcelマクロやPythonでの業務自動化スキルを社内で活かし、DX推進プロジェクトのメンバーに抜擢。

### 事例3: 物販の数値管理力が営業成績に直結

物販副業で培った「利益率計算」「在庫回転率」などの数値管理スキルが、本業の営業活動に活きた。提案書に数字を入れる習慣がつき、受注率が向上。

### 事例4: 動画編集スキルで社内プレゼンが変わった

動画編集を副業でやっていた人が、社内プレゼンに動画を取り入れるようになった。「あの人のプレゼンは分かりやすい」と評判になり、重要案件のプレゼン担当に指名されるように。

## 転職市場での副業経験の評価

転職エージェントの記事を複数読んだところ、副業経験の評価は二面性がある。

### プラス評価

| 評価ポイント | 理由 |
|------------|------|
| 「自ら学び、行動する人」 | 主体性の証明になる |
| 本業以外のスキルセット | 多角的な視点を持っている |
| マルチタスク能力 | 本業と副業の両立は時間管理力の証明 |
| 実績が見える | ブログのPV、売上など具体的な数字がある |
| 起業家精神 | 自分でビジネスを回した経験は高評価 |

### マイナス評価の可能性

| 懸念点 | 対策 |
|--------|------|
| 「本業に集中していないのでは」 | 「本業の成績にも影響はなかった」と数字で示す |
| 副業の内容が怪しい | まっとうな副業であることを説明できるようにする |
| 「すぐ辞めて副業に専念するのでは」 | 「御社でのキャリアを最優先に考えている」と伝える |

### 面接での副業の伝え方

| OK | NG |
|----|-----|
| 「ブログ運営を通じて、マーケティングの実践的なスキルを身につけました」 | 「副業で月20万円稼いでます」（お金の話を前面に出さない） |
| 「副業で得た分析力を、御社の事業でも活かしたいです」 | 「本業が暇だったので副業を始めました」 |

## 副業をキャリアに活かす戦略

### 戦略1: 本業と関連する副業を選ぶ

シナジーが生まれやすい組み合わせ。

| 本業 | 相性の良い副業 |
|------|-------------|
| 営業 | ライティング、コンサル、物販 |
| 経理・財務 | FP相談、ブログ（マネー系）、簿記講師 |
| エンジニア | プログラミング受託、技術ブログ |
| 人事 | キャリアコンサル、ライティング |
| マーケティング | ブログ、SNS運用代行 |

### 戦略2: 成果を数字で記録する

キャリアに活かすなら、副業の実績を定量的に記録しておく。

| 副業 | 記録すべき数字 |
|------|-------------|
| ブログ | 月間PV、記事数、広告収入、検索順位 |
| 物販 | 月商、利益率、取扱い商品数 |
| 動画編集 | 制作本数、リピート率、クライアント数 |
| プログラミング | 受注件数、単価の推移 |

### 戦略3: ポートフォリオを作る

副業の実績をまとめたポートフォリオを作っておくと、転職時に強力な武器になる。

- ブログならURL
- デザインなら作品集
- プログラミングならGitHub
- 動画編集なら作品リール

### 戦略4: SNSで実績を発信する

X（Twitter）やLinkedInで副業の学びや実績を発信しておくと、ヘッドハンティングのきっかけになることもある。ただし本名・顔出しは会社にバレるリスクと天秤にかける。

## まとめ

副業は「収入」と「スキル」の両方を得られる。ただし、キャリアに活かすなら「本業との関連性」を意識して副業を選ぶべき。

自分の場合、このブログ運営でライティングとマーケティングのスキルを身につけることを意識している。将来の転職で「Webマーケ」方面にも選択肢が広がればいいと思っている。
