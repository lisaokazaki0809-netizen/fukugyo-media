「資格を取れば年収が上がる」とよく言われるが、資格の種類は膨大で、どれがコスパ良いのか分からなかった。

取得にかかる時間と費用、実際の年収への効果を調べて、30代会社員の目線でランク付けした。

## 評価基準

以下の3軸で評価した。

| 評価軸 | 内容 |
|--------|------|
| 取得コスト | 勉強時間と費用 |
| 年収への効果 | 転職・昇進でどれだけ有利になるか |
| 汎用性 | 業界を問わず使えるか |

## Sランク: 高難易度だがリターンが大きい資格

### 公認会計士

| 項目 | 内容 |
|------|------|
| 取得期間 | 2から3年（社会人は3年以上が現実的） |
| 費用 | スクール代50万から80万円 + 受験料 |
| 合格率 | 約10% |
| 平均年収 | 800万から1,200万円 |

会計のプロフェッショナル資格。監査法人に就職すれば初任給でも年収500万円以上。5年経験を積めば1,000万円も見える。

ただし、勉強時間は3,000から5,000時間必要と言われており、社会人が働きながら取るのは至難の業。本気で人生を変えたい人向け。

### 中小企業診断士

| 項目 | 内容 |
|------|------|
| 取得期間 | 1から2年 |
| 費用 | スクール代20万から50万円 |
| 合格率 | 1次約30% / 2次約20% |
| 平均年収 | 独立で600万から1,000万円以上 |

経営コンサルタントの国家資格。企業内診断士として本業に活かす人も多い。

転職市場でも評価が高く、特に経営企画、事業開発、コンサルティング系のポジションで強い。副業としてコンサルティングをする人も増えている。

### USCPA（米国公認会計士）

| 項目 | 内容 |
|------|------|
| 取得期間 | 1から2年 |
| 費用 | スクール代40万から80万円 + 受験料（渡航費含め30万円程度） |
| 合格率 | 各科目50%前後 |
| 効果 | 外資系転職で年収100万から200万円アップ |

日本の公認会計士より取得しやすいが、外資系企業や海外駐在を目指す人に特に有効。英語力の証明にもなる。

## Aランク: 実用的で効果も高い資格

### TOEIC 800点以上

| 項目 | 内容 |
|------|------|
| 取得期間 | 半年から1年（現在のスコアによる） |
| 費用 | 参考書代5,000から10,000円 + 受験料7,810円 |
| 効果 | 外資系・グローバル企業で年収50万から150万円アップ |
| 資格手当 | 月5,000から20,000円（会社による） |

最もコスパが良い資格のひとつ。勉強コストが低く、効果が広い。

TOEICスコアと平均年収の関係（複数調査の平均値）:
| スコア | 平均年収 |
|--------|---------|
| 400点未満 | 約420万円 |
| 600点台 | 約480万円 |
| 800点台 | 約550万円 |
| 900点以上 | 約600万円以上 |

もちろんTOEICだけで年収が決まるわけではないが、相関は明確にある。

### 簿記2級

| 項目 | 内容 |
|------|------|
| 取得期間 | 3から6ヶ月 |
| 費用 | 参考書代3,000から5,000円 + 受験料4,720円 |
| 合格率 | 約20から25% |
| 効果 | 経理・財務への転職パスポート |

経理・財務職への転職を考えるなら必須の資格。取得コストが低いのに、転職市場での効果が大きい。

経理職の求人では「簿記2級以上」が応募条件になっていることが多い。持っていないと書類選考で落とされるレベル。

### 基本情報技術者 / 応用情報技術者

| 項目 | 内容 |
|------|------|
| 取得期間 | 3から6ヶ月 |
| 費用 | 参考書代3,000から5,000円 + 受験料7,500円 |
| 合格率 | 基本約25% / 応用約25% |
| 効果 | IT業界への転職パスポート。資格手当月5,000から30,000円 |

IT業界への転職を考えるなら、最低限の基本情報は取っておきたい。応用情報まで取れれば、かなりのアドバンテージになる。

IT企業では資格手当が充実しているところが多い。基本情報で月5,000円、応用情報で月10,000から30,000円という会社もある。

### FP2級（ファイナンシャルプランナー）

| 項目 | 内容 |
|------|------|
| 取得期間 | 3から6ヶ月 |
| 費用 | 参考書代3,000から5,000円 + 受験料8,700円 |
| 合格率 | 学科約40% / 実技約50% |
| 効果 | 金融・保険業界で評価。自分のお金の知識にもなる |

金融や保険業界への転職に有利。それ以外の業界でも「お金に詳しい人」として社内で評価されることがある。

副業との相性も良い。FPの知識を活かしてお金に関するブログやコンサルティングをする人もいる。

## Bランク: 特定分野で効果がある資格

| 資格 | 取得期間 | 費用目安 | 効果 |
|------|---------|---------|------|
| 宅地建物取引士 | 6ヶ月から1年 | 参考書+受験料で1万円 | 不動産業界で手当月2から3万円 |
| 社会保険労務士 | 1から1.5年 | スクール15から30万円 | 人事・総務で高評価 |
| AWS認定資格 | 1から3ヶ月 | 受験料3万円前後 | クラウド人材として需要大 |
| ITパスポート | 1から2ヶ月 | 参考書+受験料で5,000円 | IT系以外でもDXリテラシーの証明 |
| Google広告認定 | 1から2週間 | 無料 | Webマーケティング転職に使える |

## 取っても年収に影響しにくい資格

調べていて「これは取得コストに見合わない」と感じたもの。

- **MOS（Microsoft Office Specialist）**: 実務では「使えること」が重要で、資格自体は転職で加点されにくい
- **秘書検定**: ビジネスマナーの学習としては良いが、年収への直接的な影響はほぼない
- **色彩検定**: デザイナー以外には評価されにくい
- **漢字検定**: 趣味の領域。転職市場では評価されない
- **英検（2級以下）**: TOEICのほうが転職市場での認知度が高い

## 30代の資格戦略: 3つの原則

### 原則1: 今の業界で評価される資格を最優先

業界を変えない前提なら、業界特化の資格が最もコスパが良い。IT業界なら情報処理技術者、金融なら簿記やFP、不動産なら宅建。

### 原則2: 汎用性の高い資格を並行する

業界を変えるかもしれないなら、TOEICと簿記は鉄板。どの業界でも評価される。

### 原則3: 取得まで1年以内のものを選ぶ

2年以上かかる資格はモチベーション維持が難しい。30代は仕事も家庭も忙しい。短期で結果が出るものから着手する。

## まとめ: 自分の戦略

自分の場合、まずはこの順番で進めることにした。

1. **TOEIC 800点**（現在600点台 → 半年で目標）: コスパ最強。参考書代だけで始められる
2. **簿記2級**（並行して3ヶ月）: お金の知識は副業にも活きる
3. **その後は状況次第**: 転職を本格的に考えるなら、応用情報か中小企業診断士

資格は「取れば自動的に年収が上がる」ものではない。あくまで転職や昇進の「加点要素」。でも、その加点が50万円、100万円の差になることもある。
