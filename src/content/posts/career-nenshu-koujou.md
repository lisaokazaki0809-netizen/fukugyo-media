30代で年収を上げたい。転職だけが方法ではないはず。現実的な選択肢を調べた。

## 方法1: 社内で昇進する

最もリスクが低い方法。ただし時間がかかる。

### 昇給の実態

| 昇給タイプ | 金額目安（月額） | 年収への影響 |
|----------|-------------|-----------|
| 定期昇給（ベースアップ） | 3,000から10,000円 | 年3.6万から12万円 |
| 役職手当（係長クラス） | 30,000から50,000円 | 年36万から60万円 |
| 役職手当（課長クラス） | 50,000から100,000円 | 年60万から120万円 |

定期昇給だけで年収100万円アップは10年以上かかる。課長に昇進すれば一気に実現できるが、ポストが空くかは運もある。

### 昇進を早めるための行動

| 行動 | 効果 |
|------|------|
| 上司の期待値を確認する | 評価基準を明確にし、そこにリソースを集中 |
| 社内で目立つプロジェクトに手を挙げる | 「あの人に任せたい」と思われる |
| 部門横断の仕事を引き受ける | マネジメント能力の証明になる |
| 資格を取る | 社内の昇進要件に入っている場合あり |
| 上司に「昇進したい」と伝える | 意外とこれをやっていない人が多い |

## 方法2: 転職する

30代前半なら、同業界への転職で年収10から20%アップが現実的なライン。

### 転職での年収アップシミュレーション

| 現年収 | 10%アップ | 20%アップ | 30%アップ（難易度高） |
|--------|---------|---------|-----------------|
| 400万円 | 440万円 | 480万円 | 520万円 |
| 500万円 | 550万円 | 600万円 | 650万円 |
| 600万円 | 660万円 | 720万円 | 780万円 |
| 700万円 | 770万円 | 840万円 | 910万円 |

年収500万円から20%アップの600万円なら、転職1回で100万円アップは可能。

### 転職のリスク

| リスク | 対策 |
|--------|------|
| 新しい環境への適応ストレス | 入社前に社風をしっかり調べる |
| 人間関係のリセット | 最初の3ヶ月は信頼構築に集中 |
| 期待と現実のギャップ | 面接で具体的な業務内容を確認 |
| 試用期間中の不安 | 前職の成功体験を再現する |

## 方法3: 副業する

本業の年収はそのままで、副業で年間100万円を上乗せする。月換算で約8.3万円。

### 副業の種類別: 月8万円の達成難易度

| 副業 | 月8万円の達成難易度 | 必要期間 |
|------|---------------|---------|
| Webライティング | 中 | 3から6ヶ月 |
| 動画編集 | 中 | 3から6ヶ月 |
| プログラミング | 高 | 6ヶ月から1年 |
| ブログ・アフィリエイト | 非常に高 | 1年以上 |
| せどり・物販 | 中 | 2から4ヶ月 |
| コンサルティング | 高（実績必要） | 実績次第 |

月8万円は簡単ではないが、不可能でもない。ただし週10から15時間の作業時間が必要。

## 方法4: 資格を取って手当をもらう

会社によっては資格手当がある。複数の資格を積み重ねると、年間数十万円の上乗せになる。

### 資格手当の積み上げシミュレーション

| 資格 | 月額手当 | 取得期間 |
|------|---------|---------|
| TOEIC 800点 | 10,000円 | 半年 |
| 基本情報技術者 | 5,000円 | 3ヶ月 |
| 簿記2級 | 5,000円 | 3ヶ月 |
| FP2級 | 3,000円 | 3ヶ月 |
| 合計 | 23,000円/月 | |
| 年間 | 276,000円 | |

資格手当だけで年間約28万円。副業と組み合わせれば100万円は見えてくる。

ただし、手当の金額や対象資格は会社によって大きく異なるので、事前に就業規則で確認が必要。

## 方法5: 成果報酬型の仕事にシフトする

営業職やインセンティブがある仕事に移ることで、成果次第で年収が上がる。

| 職種 | インセンティブの目安 |
|------|---------------|
| 不動産営業 | 成約額の3から5% |
| 保険営業 | 初年度保険料の20から40% |
| 外資系営業 | ベース+成果で年収2倍になることも |
| フリーランス | 上限なし（ただしゼロもある） |

安定志向の人には向かないが、実力に自信があるなら検討の価値あり。

## 5つの方法を総合比較

| 方法 | 実現までの期間 | リスク | 再現性 | 100万円到達の難易度 |
|------|-------------|--------|--------|---------------|
| 社内昇進 | 3年以上 | 低 | 中 | 高 |
| 転職 | 3から6ヶ月 | 中 | 高 | 中 |
| 副業 | 半年から1年 | 低 | 中 | 中 |
| 資格手当 | 3ヶ月から1年 | 低 | 高 | 高（単体では100万円は難しい） |
| 成果報酬 | すぐ | 高 | 低 | 中 |

## おすすめの組み合わせ

単独で年収100万円アップは難しくても、複数を組み合わせれば現実的になる。

### パターンA: 安全重視

| 方法 | 年収アップ額 |
|------|-----------|
| 資格手当（TOEIC+簿記） | +20万円 |
| 副業（ライティング） | +60万円 |
| 定期昇給 | +10万円 |
| 合計 | +90万円 |

### パターンB: 攻め重視

| 方法 | 年収アップ額 |
|------|-----------|
| 転職 | +80万から100万円 |
| 合計 | +80万から100万円 |

### パターンC: 長期安定

| 方法 | 年収アップ額 |
|------|-----------|
| 昇進（係長→課長） | +60万円 |
| 資格手当 | +20万円 |
| 副業（ブログ） | +20万円 |
| 合計 | +100万円 |

## まとめ

一番現実的なのは「転職」か「資格手当+副業」の組み合わせ。転職は1回で100万円アップが見込めるし、資格と副業の積み上げでも到達可能。

自分は今のところ、副業を育てつつ資格も取る方針。転職は本当に必要になったときの切り札として取っておく。
