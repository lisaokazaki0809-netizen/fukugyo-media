メルカリで不用品を売った経験はあるけど、「せどり」として継続的に稼ぐことはできるのか調べた。

## せどりの基本

安く仕入れて高く売る。シンプルだけど、利益計算をしっかりやらないと赤字になる。

### 仕入れ先の種類と特徴

| 仕入れ先 | メリット | デメリット |
|---------|---------|----------|
| ブックオフなど中古ショップ | 掘り出し物がある。値札で利益計算しやすい | 足を運ぶ必要がある。良い商品は競合も狙っている |
| ドンキ・家電量販店 | セール品が狙い目。ポイント還元も利益になる | 大型商品は送料がかさむ |
| Amazon・楽天のセール | 自宅で仕入れ完結。タイムセールが狙い目 | 価格差が小さいことが多い |
| 海外通販（AliExpress等） | 仕入れ値が圧倒的に安い | 届くまで2から4週間。品質リスクあり |
| 卸売サイト（NETSEA等） | 業者価格で仕入れ可能 | 最低ロットがある。在庫リスク |

### 売れやすいジャンル

調べた限り、メルカリで安定して売れるジャンル。

| ジャンル | 特徴 | 利益率の目安 |
|---------|------|-----------|
| ブランド品 | 単価が高く利益も大きい | 20から40% |
| ゲーム・おもちゃ | 限定品は高値がつく | 15から30% |
| 書籍・参考書 | 仕入れが安い。送料も安い | 30から50% |
| アパレル | 回転が速い。トレンドに左右される | 20から40% |
| コスメ・美容 | 未開封品が高値。リピーター多い | 20から35% |
| 家電・ガジェット | 単価が高い。型落ちが狙い目 | 10から25% |

## 利益計算を具体的にやってみた

メルカリで商品を売るとき、手元に残る利益は思ったより少ない。

### ケース1: 書籍

| 項目 | 金額 |
|------|------|
| 仕入れ値（ブックオフ） | 200円 |
| メルカリ販売価格 | 1,200円 |
| メルカリ手数料（10%） | -120円 |
| 送料（ゆうメール） | -180円 |
| 利益 | 700円 |
| 利益率 | 58% |

### ケース2: ゲームソフト

| 項目 | 金額 |
|------|------|
| 仕入れ値（セール品） | 2,500円 |
| メルカリ販売価格 | 4,500円 |
| メルカリ手数料（10%） | -450円 |
| 送料（らくらくメルカリ便） | -210円 |
| 梱包材 | -50円 |
| 利益 | 1,290円 |
| 利益率 | 29% |

### ケース3: 仕入れミス（赤字パターン）

| 項目 | 金額 |
|------|------|
| 仕入れ値 | 3,000円 |
| メルカリ販売価格（売れずに値下げ） | 2,500円 |
| メルカリ手数料（10%） | -250円 |
| 送料 | -380円 |
| 利益 | -1,130円（赤字） |

売れない商品は値下げせざるを得ない。仕入れ判断を間違えると簡単に赤字になる。

## どれくらい稼げるのか

| レベル | 月の作業時間 | 月商 | 月の利益 | 時給換算 |
|--------|-----------|------|---------|---------|
| 初心者（不用品販売の延長） | 10時間 | 3万から5万円 | 1万から2万円 | 1,000から2,000円 |
| 中級者（仕入れルート確立） | 20から30時間 | 10万から20万円 | 3万から8万円 | 1,500から2,700円 |
| 上級者（外注化・効率化） | 40時間以上 | 30万円以上 | 10万円以上 | 2,500円以上 |

初心者が月1万から2万円稼ぐのは現実的。ただし時給換算すると最低賃金を下回ることも多い。

## せどりの注意点

### 在庫リスク

売れない商品は不良在庫になる。部屋が商品で溢れかえる。精神的にもキツいし、仕入れ資金が回収できないリスクがある。

### メルカリの規約

違反するとアカウント停止のリスクがある。

| 禁止行為 | 内容 |
|---------|------|
| 無在庫転売 | 手元にない商品を出品する行為 |
| チケット転売 | 法律でも禁止されている |
| 偽ブランド品 | 犯罪行為 |
| 医薬品・化粧品（一部） | 未開封でも出品できないカテゴリがある |
| 同一商品の大量出品 | 業者とみなされてアカウント制限される場合がある |

### 確定申告

せどりで年間20万円以上の所得（売上-経費）がある場合、確定申告が必要。

経費として計上できるもの:
- 仕入れ代金
- 送料
- 梱包材
- 交通費（仕入れのための移動）
- スマホ代の一部（按分）
- 有料ツールの利用料

## せどりに向いている人・向いていない人

| 向いている人 | 向いていない人 |
|------------|-------------|
| 買い物が好き | 梱包・発送が面倒に感じる |
| 相場観がある | 在庫を置くスペースがない |
| コツコツ作業ができる | 「楽に稼ぎたい」と思っている |
| 数字に強い | 利益計算が苦手 |

## まとめ

メルカリせどりは今でも稼げる副業だが、利益計算と在庫管理をしっかりやらないと赤字になる。

まずは家の不用品を売ることから始めて、相場感を掴むのが最も安全なスタート。いきなり仕入れに何万円も使うのは危険。

自分もまずは不用品販売からやってみて、感覚が掴めたら少額の仕入れに挑戦してみようと思う。
