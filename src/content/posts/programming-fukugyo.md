「プログラミングを学んで副業で稼ごう」という広告をよく見る。未経験からどれくらい現実的なのか調べた。

## プログラミングスクールの実態

まず、スクールの広告を鵜呑みにしていいのか調べた。

### 広告と実態の比較

| 広告の謳い文句 | 実態 |
|-------------|------|
| 「3ヶ月で副業デビュー」 | 基礎は学べるが実務レベルには遠い |
| 「未経験から月10万円」 | さらに半年から1年の実務経験が必要 |
| 「案件獲得保証」 | 超低単価の案件を紹介するだけの場合あり |
| 「転職保証」 | 転職先がSES（客先常駐）のみの場合が多い |

### スクールの費用

| スクール形態 | 費用 | 期間 |
|------------|------|------|
| オンライン独学型 | 月1万から3万円 | 3から6ヶ月 |
| メンター付きオンライン | 15万から40万円 | 3から6ヶ月 |
| 通学型スクール | 30万から80万円 | 3から6ヶ月 |
| 転職保証型 | 40万から70万円 | 3から6ヶ月 |

高額なスクールが良いとは限らない。独学+メンター（MENTAなどで月1万から3万円）で十分学べるという声も多い。

### 無料で学べるリソース

| サービス | 特徴 |
|---------|------|
| Progate | ゲーム感覚で基礎が学べる。有料版は月1,078円 |
| ドットインストール | 3分動画で効率的。プレミアムは月1,080円 |
| freeCodeCamp | 英語だが完全無料。カリキュラムが体系的 |
| YouTube | 無料の講座が大量にある |
| Udemy | セール時は1,500円程度で質の高い講座が買える |

月1,000から2,000円で十分な学習環境が手に入る。いきなり30万円以上のスクールに入る必要はない。

## 未経験者が受注できる案件

クラウドワークスやランサーズで「未経験OK」の案件を調べた。

### 案件タイプ別の相場と実態

| 案件 | 単価 | 作業時間（初心者） | 時給換算 | 競合 |
|------|------|---------------|---------|------|
| HTML/CSSコーディング | 5,000から30,000円/ページ | 10から20時間 | 250から1,500円 | 非常に多い |
| WordPress構築 | 30,000から100,000円/サイト | 30から60時間 | 500から1,667円 | 多い |
| LP制作 | 50,000から200,000円 | 30から50時間 | 1,000から4,000円 | 中程度 |
| Webアプリ開発 | 100,000から500,000円 | 50から100時間 | 1,000から5,000円 | 少ない |

### 時給換算の現実

未経験者がLP制作の案件（5万円）を受けた場合。

| 工程 | 時間 |
|------|------|
| 学習（HTML/CSS/JS基礎） | 100時間 |
| デザインの理解 | 20時間 |
| 実際のコーディング | 30時間 |
| 修正対応 | 10時間 |
| 合計 | 160時間 |

160時間で5万円。時給313円。最低賃金を大きく下回る。

ただし、2件目以降は学習時間がなくなるので、作業時間は40から50時間に短縮される。時給は1,000から1,250円まで上がる。

## 稼げるようになるまでのタイムライン

| 期間 | 状態 | 月収の目安 |
|------|------|----------|
| 1から3ヶ月 | 基礎学習。Progate、ドットインストールで毎日1から2時間 | 0円 |
| 4から6ヶ月 | 初案件に挑戦。ポートフォリオを作る | 0から3万円 |
| 7から9ヶ月 | 2から3件目の案件。スピードが上がってくる | 3から5万円 |
| 10から12ヶ月 | リピーターがつき始める | 5から10万円 |
| 1年以上 | 指名案件が来る。単価も上がる | 10万円以上 |

最初の6ヶ月はほぼ無収入で勉強する期間。ここを乗り越えられるかが分かれ道。

## どの言語を学ぶべきか

副業で稼ぐことが目的なら、以下の順がおすすめらしい。

| 優先度 | 言語/技術 | 理由 |
|--------|---------|------|
| 1 | HTML/CSS | Web制作の基本。これなしでは何もできない |
| 2 | JavaScript | Webサイトに動きをつける。必須スキル |
| 3 | WordPress | 案件数が圧倒的に多い |
| 4 | React or Vue | 高単価のフロントエンド案件に必要 |
| 5 | PHP or Python | バックエンド。ここまで来れば月20万円以上も |

最初の3つ（HTML/CSS/JS/WordPress）だけで、月5から10万円の案件は受けられる。

## プログラミング以外の選択肢との比較

同じ時間を投資するなら、他のスキルのほうが効率的な場合もある。

| スキル | 学習期間 | 初案件までの時間 | 月収の目安 |
|--------|---------|-------------|----------|
| Webライティング | 2週間 | 1ヶ月 | 1から5万円 |
| 動画編集 | 1ヶ月 | 1から2ヶ月 | 3から10万円 |
| Webデザイン | 2から3ヶ月 | 3ヶ月 | 3から15万円 |
| プログラミング | 3から6ヶ月 | 6ヶ月 | 5から30万円 |

プログラミングは学習コストが最も高いが、長期的には単価が最も上がりやすい。「すぐ稼ぎたい」ならライティングや動画編集、「長期的にスキルを積みたい」ならプログラミング。

## まとめ

プログラミング副業は、未経験から月10万円稼ぐのに1年はかかる。スクール費用も考えると、投資回収にさらに時間がかかる。

「プログラミングが好き」「長期的にスキルを身につけたい」なら良い選択肢。でも「手っ取り早く稼ぎたい」なら向いていない。

自分はこのブログを運営しながらWeb周りの知識を少しずつ身につける方針にした。いきなりスクールに30万円払うのではなく、まずは無料のProgateから始めて適性を見極める。
