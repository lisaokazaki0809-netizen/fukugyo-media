転職面接で何を聞かれるのか、事前に調べて準備しておきたかったのでまとめた。

## 30代の面接で必ず聞かれる質問5つ

### 1. 転職理由

「なぜ今の会社を辞めたいのか」。面接官が最も重視する質問。

| NGな回答 | 理由 | 改善例 |
|---------|------|--------|
| 「人間関係が悪い」 | 愚痴に聞こえる | 「チームで成果を出す環境で働きたい」 |
| 「給料が低い」 | お金だけで動く人に見える | 「成果が正当に評価される環境を求めている」 |
| 「仕事がつまらない」 | 受け身に見える | 「より専門性の高い業務に挑戦したい」 |
| 「残業が多い」 | ワークライフバランスだけが動機に見える | 「生産性の高い環境で、質の高い仕事をしたい」 |

ポイントは「ネガティブな理由をポジティブに変換する」こと。「今の環境ではできないことがある。御社ならそれができる」という構成にする。

### 2. これまでの実績

30代は「何ができるか」を具体的に求められる。職務経歴書に書いた数字を、口頭でも語れるように準備する。

| 弱い回答 | 強い回答 |
|---------|---------|
| 「営業をやっていました」 | 「法人営業として年間50社を担当し、3年連続で目標達成率120%以上でした」 |
| 「プロジェクトに参加していました」 | 「5名のチームリーダーとして、3ヶ月で新サービスを立ち上げ、初年度売上3,000万円を達成しました」 |

### 3. マネジメント経験

30代になると聞かれる確率が格段に上がる。

リーダー経験がなくても、広い意味でのマネジメント経験をアピールできる。

| 経験 | 伝え方 |
|------|--------|
| 後輩の指導 | 「新人2名のOJTを担当し、6ヶ月で独り立ちさせました」 |
| プロジェクトの取りまとめ | 「3部門横断のプロジェクトで、スケジュール管理と進捗報告を担当しました」 |
| 外注先の管理 | 「外部ベンダー3社との折衝と品質管理を行い、納期遅延ゼロを達成しました」 |

### 4. 5年後のキャリアビジョン

「御社でどうなりたいか」を聞いている。会社の事業内容を調べた上で、具体的に答える必要がある。

| NGな回答 | 改善例 |
|---------|--------|
| 「特に考えていません」 | 「3年で事業部のマネージャーを目指し、5年後にはP&L責任を持てるポジションで事業成長に貢献したいです」 |
| 「独立したいです」 | 「御社の成長フェーズで経験を積み、経営視点を持ったマネージャーになりたいです」 |

### 5. 年収の希望

現年収をベースに10から20%アップが現実的なライン。

| 現年収 | 交渉レンジ | 伝え方 |
|--------|----------|--------|
| 500万円 | 550から600万円 | 「現年収は500万円です。経験とスキルを考慮いただき、550万円以上を希望しています」 |
| 700万円 | 770から840万円 | 「現年収は700万円です。御社での貢献度に見合った評価をいただければ幸いです」 |

根拠なく高い金額を言うと心証が悪い。「なぜその金額が妥当か」を説明できるようにしておく。

## 逆質問で好印象を与える

面接の最後に「何か質問はありますか？」と聞かれる。ここで良い質問ができると評価が上がる。

### 好印象を与える逆質問

| 質問 | 意図 |
|------|------|
| 「入社後、最初の3ヶ月で期待される成果は何ですか？」 | 具体的に貢献したいという姿勢を見せる |
| 「この部署の今後1年の目標を教えてください」 | 事業への関心を示す |
| 「活躍されている方に共通する特徴はありますか？」 | 自分もそうなりたいという意欲を示す |
| 「チームの課題があれば教えてください」 | 課題解決に貢献したいという姿勢 |

### NGな逆質問

| 質問 | 理由 |
|------|------|
| 「残業はどれくらいですか？」 | 最初に聞くと「仕事より条件」に見える |
| 「有給は取れますか？」 | 同上 |
| 「研修制度はありますか？」 | 受け身な印象を与える |
| 「特にありません」 | 関心がないと思われる |

残業や有給については、内定後の条件交渉フェーズで聞くのが正解。

## 面接の準備チェックリスト

| 準備項目 | 確認 |
|---------|------|
| 応募企業の事業内容、直近のニュース、IR情報を調べたか | |
| 自分の職務経歴を時系列で整理したか | |
| 各プロジェクトの成果を数字で表現できるか | |
| 転職理由を前向きな表現で言語化したか | |
| 5年後のキャリアビジョンを企業に合わせて用意したか | |
| 逆質問を3つ以上用意したか | |
| スーツ・身だしなみは整っているか | |
| 面接会場の場所と所要時間を確認したか | |

## オンライン面接の注意点

コロナ以降、一次面接がオンラインになることが増えている。

| ポイント | 内容 |
|---------|------|
| 背景 | 白壁やバーチャル背景。生活感のある背景はNG |
| 照明 | 顔が暗くならないようにデスクライトを正面に |
| 音声 | イヤホンマイク使用。部屋のエコーを防ぐ |
| 目線 | カメラを見る（画面ではなく）。目線が下がると印象が悪い |
| 服装 | 上半身はスーツ。下半身も念のため |
| 通信環境 | 有線LANが安全。Wi-Fiなら事前にスピードテスト |

## まとめ

面接は準備が9割。30代の転職は「経験と実績」を問われるので、自分の仕事を棚卸しして数字で語れるようにしておくことが重要。
