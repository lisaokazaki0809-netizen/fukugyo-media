「まず1円でもいいから副業で稼いでみよう」と思い、クラウドワークスに登録して初案件を受注した。その全記録。

## 登録から案件探しまで

### クラウドワークスに登録（10分）

メールアドレスとプロフィールを入力するだけ。本人確認書類の提出は後でもできる。

プロフィールは以下を書いた。
- 職業: 会社員（副業）
- スキル: ライティング、リサーチ
- 自己紹介: 30代会社員。副業として記事作成に取り組んでいます。正確な情報収集と分かりやすい文章を心がけます。

### 案件を探す

最初は「初心者歓迎」で絞って検索した。

| 案件タイプ | 件数 | 単価 |
|----------|------|------|
| 記事作成（1,000文字） | 多数 | 500から2,000円 |
| アンケート回答 | 大量 | 10から100円 |
| データ入力 | やや多い | 1件50から200円 |
| 商品レビュー | そこそこ | 500から1,000円 |

アンケートやデータ入力は単価が低すぎて割に合わない。記事作成に絞った。

## 提案文を送る

気になる案件に提案文を送った。最初に送った3件は全部スルーされた。

### 落ちた提案文（テンプレ感丸出し）

> はじめまして。記事作成に興味があり、応募させていただきました。丁寧な記事を心がけます。よろしくお願いいたします。

これでは選ばれない。クライアントは毎日大量の提案文を受け取っている。差別化できないテンプレは読まれもしない。

### 受かった提案文（具体的に書いた）

> はじめまして。副業メディアを運営しており、投資・副業ジャンルの記事を20本以上執筆しています。
>
> 本案件のテーマについて、以下の構成で執筆できます。
> 1. 導入（読者の悩みに共感）
> 2. 本題（具体的な数字やデータを入れる）
> 3. まとめ（読者の行動につながる結び）
>
> 納期は3日以内で対応可能です。テストライティングも承ります。

具体的な構成案を入れたこと、自分のブログがあることが評価されたらしい。

## 初案件の内容

| 項目 | 内容 |
|------|------|
| テーマ | 「30代の転職で気をつけること」に関する記事 |
| 文字数 | 2,000文字 |
| 報酬 | 1,500円（税込） |
| 納期 | 5日以内 |
| 修正 | 2回まで無料 |

正直、文字単価0.75円は安い。でも「最初の実績を作る」ことが目的なので、単価は気にしないことにした。

## 実際の作業

| 工程 | 時間 |
|------|------|
| リサーチ | 40分 |
| 構成案作成 | 15分 |
| 本文執筆 | 1時間20分 |
| 見直し・修正 | 20分 |
| 合計 | 2時間35分 |

2,000文字の記事に2時間半。時給換算すると約580円。最低賃金を下回っている。

ただ、自分のブログで同じテーマの記事を書いたことがあったので、リサーチの時間は短くできた。ブログ運営が案件にも活きている。

## 納品と修正

初稿を納品したら、以下の修正依頼が来た。

- 見出しのh2タグが抜けている箇所がある → 修正
- 「です・ます調」に統一してほしい → 修正
- 最後にCTA（行動を促す文章）を追加してほしい → 追加

修正は30分で完了。2回目の修正は発生せず、検収完了。

## 報酬の受け取り

| 項目 | 金額 |
|------|------|
| 報酬額 | 1,500円 |
| システム手数料（20%） | -300円 |
| 手取り | 1,200円 |

手数料20%は痛い。1,500円の案件だと手取りは1,200円。ここからさらに振込手数料が100から500円かかる（楽天銀行なら100円）。

### 実質の手取り

- 報酬: 1,500円
- システム手数料: -300円
- 振込手数料: -100円
- 実質手取り: 1,100円
- 作業時間: 2時間35分 + 修正30分 = 3時間5分
- 時給: 約357円

厳しい現実。でも「副業で初めてお金を稼いだ」という事実は、金額以上の価値があった。

## 初案件で学んだこと

| 学び | 詳細 |
|------|------|
| 提案文が命 | テンプレでは絶対に受からない。具体的な構成案を入れる |
| 自分のブログが武器になる | 「実績」として提示できる。ブログやっていてよかった |
| 文字単価1円未満は修行 | 実績のための投資と割り切る。3件こなしたら単価を上げる |
| 修正は当たり前 | 最初から完璧は無理。修正を嫌がらない姿勢が大事 |
| 手数料がエグい | クラウドソーシングの20%は大きい。慣れたら直接契約に移行すべき |

| 目標 | 期限 |

| あと2件受注して実績を3件にする | 2週間以内 |
