このブログを立ち上げて約1ヶ月が経った。「副業ブログって実際どうなの」という疑問に対して、自分の数字をそのまま公開する。

## 1ヶ月の数字

| 指標 | 数字 |
|------|------|
| 記事数 | 21本 |
| 総PV | 約340PV |
| ユニークユーザー | 約180人 |
| 収益 | 0円 |
| 作業時間 | 約60時間 |
| 時給換算 | 0円 |

収益ゼロ。分かっていたけど、改めて数字で見ると虚しい。

## PVの内訳

| 流入元 | PV | 割合 |
|--------|-----|------|
| SNS（X） | 約200 | 59% |
| 直接アクセス | 約80 | 24% |
| 検索（Google） | 約40 | 12% |
| その他 | 約20 | 5% |

検索からの流入はほぼない。ブログを始めて1ヶ月では、Googleにインデックスされている記事も少ないし、検索順位もまだまだ。

SNSからの流入が6割。Xで記事を投稿するたびに少しだけアクセスが来る。ただし一過性で、翌日にはほぼゼロに戻る。

## 時間の使い方

| 作業 | 時間 | 割合 |
|------|------|------|
| 記事の調査・執筆 | 40時間 | 67% |
| サイトのデザイン・構築 | 15時間 | 25% |
| SNS投稿・運用 | 5時間 | 8% |
| 合計 | 60時間 | |

1記事あたり平均2時間弱。調査に時間がかかる記事は3時間以上。

## やってよかったこと

### 1. 最初にサイト設計をちゃんとやった

カテゴリ分け、記事の方向性、ターゲットを最初に決めておいたおかげで、記事を書くときに迷わなかった。

### 2. SEOの基盤を最初から入れた

sitemap、構造化データ、meta descriptionなど、後から入れると面倒なものを最初に設定した。これは正解だった。

### 3. 「調べた」ベースで書いた

体験談がないのに「やってみた」と嘘をつくより、正直に「調べた」と書いたほうが気が楽。実際に体験したことは後から追記できる。

## 後悔していること

### 1. 最初から記事数を出しすぎた

21本を一気に公開したが、日付が近すぎて不自然。週1から2本ペースで公開したほうが、Googleからの評価も分散できたかもしれない。

### 2. SNS運用が足りない

Xのアカウントを作ったが、フォロワーはまだ数十人。記事を投稿しても届く範囲が狭い。もっと積極的に発信すべきだった。

### 3. 収益化の準備をしていなかった

AdSenseの申請もアフィリエイトの登録もまだやっていない。記事が溜まった今のタイミングで始めるべき。

## 2ヶ月目の目標

| 目標 | 具体的に |
|------|---------|
| 記事を5本追加 | 体験談ベースの記事を増やす |
| 月間PV 1,000 | SNSでの発信を強化 |
| AdSense申請 | 記事数は足りているはず |
| アフィリエイト登録 | A8.net、もしもアフィリエイトに登録 |
| 検索流入を増やす | Search Consoleでインデックス状況を確認 |

## 結論: 1ヶ月目は「種まき」

収益ゼロは想定内。ブログの収益化は早くても半年、一般的には1年かかると言われている。

1ヶ月目に大事なのは「続けられる仕組みを作ること」。記事のストックができて、デザインも整って、SEOの基盤もできた。種は蒔いた。あとは芽が出るのを待ちながら、水をやり続けるだけ。

3ヶ月目にまた経過報告を書く。
