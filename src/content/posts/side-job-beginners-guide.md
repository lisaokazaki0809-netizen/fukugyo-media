副業を始めようと思い立ったものの、何から手をつければいいのか分からなかった。

ネットで情報を集めてみると、「月収100万円」みたいな煽り記事ばかりで、基本的なことを堅実にまとめたページが意外と少ない。

とりあえず動く前に、最低限これだけは調べておこうと思ったことを5つ書いておく。

## 1. 会社の就業規則を確認した

副業を始めるなら、まず自分の会社の就業規則を確認するのが第一歩。

### 就業規則のパターン

| パターン | 内容 | 割合（推定） |
|---------|------|------------|
| 全面禁止 | 副業そのものが禁止 | 約30% |
| 届出制 | 会社に届け出れば副業可能 | 約40% |
| 許可制 | 会社の許可が必要 | 約20% |
| 制限なし | 自由に副業できる | 約10% |

2018年に厚生労働省がモデル就業規則を改定し、副業・兼業を原則容認する方向になった。ただし、全部の会社がこれに従っているわけではない。

うちの会社は「届出制」だった。届け出れば副業OK。ここがNGだったら計画を練り直す必要があったので、最初に確認してよかった。

### 就業規則に違反したらどうなるか

就業規則に違反して副業していたことがバレた場合のリスク。

- 口頭注意や始末書で済むケースが多い
- 悪質な場合（競合企業での副業など）は懲戒処分の対象になりうる
- 過去の判例では、会社の業務に支障がない副業を理由とした解雇は無効とされたケースもある

ただし、裁判になること自体が大きなリスクなので、ルールを守るに越したことはない。

## 2. 確定申告のルールを調べた

副業で年間20万円を超える所得がある場合、確定申告が必要になる。

### 「所得」と「収入」の違い

ここが一番混乱するポイント。

| 用語 | 意味 | 例 |
|------|------|-----|
| 収入 | 副業で得た総額 | 30万円 |
| 経費 | 副業に使った費用 | 15万円 |
| 所得 | 収入 - 経費 | 15万円 |

たとえば副業で30万円の収入があっても、経費が15万円かかっていれば所得は15万円。この場合、確定申告は不要。

ただし注意点がある。

- **住民税の申告は必要**: 所得が20万円以下でも、住民税の申告は別途必要
- **複数の副業がある場合**: すべての副業の所得を合算して20万円を超えるかで判断する
- **赤字の場合**: 副業が赤字でも、確定申告すれば本業の所得と相殺できることがある

### 確定申告の方法

| 方法 | 特徴 |
|------|------|
| e-Tax（スマホ） | マイナンバーカードがあれば自宅で完結 |
| e-Tax（PC） | 詳細な入力が可能 |
| 税務署に直接提出 | 分からないことをその場で聞ける |
| 税理士に依頼 | 費用は3万から10万円。所得が多い場合は検討 |

副業の所得が年間20万から50万円程度なら、e-Taxで自分でやるのが現実的。

### 会社にバレない方法

副業が会社にバレる一番の原因は「住民税」。副業の所得が加わると住民税が上がり、会社の経理に気づかれる。

対策: 確定申告時に住民税を「自分で納付（普通徴収）」にチェックする。これで副業分の住民税は自宅に届き、会社を経由しない。

ただし、すべての自治体で普通徴収に対応しているわけではない。事前に市区町村の税務課に確認する必要がある。

## 3. 使える時間を洗い出した

本業に支障が出たら本末転倒なので、現実的に使える時間を計算してみた。

### 自分のスケジュール

| 時間帯 | 使える時間 | 備考 |
|--------|----------|------|
| 平日朝（6:00-7:30） | 1時間 | 出勤前。集中力が高い |
| 平日夜（21:00-23:00） | 1から1.5時間 | 疲れ具合による |
| 土曜 | 3から4時間 | 午前中が効率的 |
| 日曜 | 2から3時間 | 休息も必要 |
| **週合計** | **10から15時間** | |

まずは週5時間を副業に充てるところから始めることにした。慣れてきたら徐々に増やす。

### 時間管理のコツ

調べていて参考になったアドバイス。

- **曜日と時間を固定する**: 「毎日21時から22時は副業の時間」と決めるとサボりにくい
- **スマホの通知をオフにする**: 副業時間中のSNS通知は生産性を破壊する
- **休息日を必ず作る**: 週7日フル稼働はすぐに燃え尽きる。最低週1日は完全オフに

## 4. 初期投資の上限を決めた

ネットを見ていると「まず50万円の教材を買いましょう」みたいな情報が出てくる。これは完全にスルー。

### 自分ルール: 初期投資は最大3万円

| 項目 | 予算 |
|------|------|
| パソコン | 持っているものを使う（0円） |
| ドメイン・サーバー | 年間1万から1.5万円 |
| 参考書・教材 | 5,000から10,000円 |
| ツール・ソフト | 無料プランで始める |
| **合計** | **1.5万から3万円** |

まっとうな副業なら、初期投資は3万円以内で始められる。それ以上の金額を要求されたら、まず疑うべき。

### 警戒すべき初期投資

- 「稼げるノウハウ教えます」系の高額教材（10万から50万円）
- 「このツールを使えば自動で稼げる」系のソフトウェア
- 「先行投資で半年後に回収」と言われるビジネス

これらは情報商材詐欺や、マルチ商法の入り口である可能性が高い。

## 5. 「すぐに稼げる」は疑うことにした

調べれば調べるほど、「1ヶ月で月収100万円」みたいな話が出てくる。

### 副業の現実的なタイムライン

| 期間 | 状態 |
|------|------|
| 1から2ヶ月目 | 学習期間。収入ゼロが普通 |
| 3から4ヶ月目 | 初収益。月数千円から1万円 |
| 5から6ヶ月目 | 軌道に乗り始める。月1万から3万円 |
| 7から12ヶ月目 | 安定してくる。月3万から10万円 |
| 1年以上 | 月10万円以上も可能 |

冷静に考えて、そんなに簡単なら会社員をやっている人はいない。最低3から6ヶ月はかかるものと覚悟して、焦らずやることにした。
