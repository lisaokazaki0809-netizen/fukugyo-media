英語ができると年収が上がるデータは多い。でも学習法がありすぎてどれがいいか分からなかったので、コスパの観点で調べた。

## 英語力と年収の関係

まず、本当に英語で年収が上がるのか確認した。

### TOEICスコアと平均年収

| TOEICスコア | 平均年収 | 500点未満との差 |
|------------|---------|-------------|
| 500点未満 | 約420万円 | - |
| 500から595点 | 約450万円 | +30万円 |
| 600から695点 | 約480万円 | +60万円 |
| 700から795点 | 約520万円 | +100万円 |
| 800から895点 | 約550万円 | +130万円 |
| 900点以上 | 約600万円以上 | +180万円以上 |

もちろんTOEICだけで年収が決まるわけではないが、相関は明確にある。TOEIC 800点を超えると、外資系やグローバル企業への転職で大きなアドバンテージになる。

### 企業の資格手当

| 企業タイプ | TOEIC基準 | 手当額 |
|----------|----------|--------|
| 大手メーカー | 730点以上 | 月5,000から15,000円 |
| 金融機関 | 800点以上 | 月10,000から20,000円 |
| IT企業 | 700点以上 | 月5,000から10,000円 |
| 外資系 | 基準なし（業務で使えることが前提） | 給与に内包 |

月1万円の手当でも、年間12万円。10年で120万円の差になる。

## 学習法の徹底比較

| 方法 | 月額費用 | メリット | デメリット | 向いている人 |
|------|---------|---------|----------|-----------|
| 英会話スクール（対面） | 2万から5万円 | 対面の緊張感、仲間ができる | 高い、通学の手間 | 人と話して学びたい人 |
| オンライン英会話 | 6,000から12,000円 | 安い、毎日受けられる | 自分で続ける意志が必要 | 会話力を伸ばしたい人 |
| 英語コーチング | 10万から20万円 | 短期集中、専属コーチ | 非常に高い | 短期で結果を出したい人 |
| 英語アプリ | 0から2,000円 | 最安、すきま時間に | スピーキングが弱い | 基礎固めしたい人 |
| 独学（参考書） | 書籍代のみ | マイペース | モチベ維持が難しい | 自律できる人 |

### コスパ最強の組み合わせ

調べた結果、以下の組み合わせが最もコスパが良い。

| ツール | 月額 | 用途 |
|--------|------|------|
| スタディサプリENGLISH | 2,178円 | TOEIC対策の基礎 |
| 金のフレーズ（書籍） | 979円（買い切り） | TOEIC頻出単語 |
| DMM英会話 or ネイティブキャンプ | 6,480から7,480円 | スピーキング練習 |
| 公式問題集（書籍） | 3,300円（買い切り） | 実践演習 |
| **月額合計** | **約9,000円** | |

月9,000円で質の高い学習環境が手に入る。英会話スクール1ヶ月分の費用で、この組み合わせなら4から5ヶ月続けられる。

## TOEIC対策の効率的な順番

### ステップ1: 単語（1から2ヶ月）

「金のフレーズ」を1冊やり切る。TOEIC頻出の1,000語を覚えるだけで、100点くらい上がる人もいるらしい。

毎日の学習ルーティン:
- 朝の通勤: 50語の復習（15分）
- 夜: 新規50語の暗記（20分）
- 1ヶ月で1,000語を1周、2ヶ月で3周

### ステップ2: 文法（1ヶ月）

中学英語の文法を総復習。スタディサプリの文法講座が分かりやすい。

### ステップ3: リスニング（継続）

オンライン英会話を毎日25分。これを3ヶ月続けると、リスニングセクションのスコアが大幅に上がる。

### ステップ4: 公式問題集（1から2ヶ月）

本番と同じ形式で時間を計って解く。2から3回分をやれば出題パターンに慣れる。

### 期間別の目標スコアアップ

| 現在のスコア | 3ヶ月後の目標 | 6ヶ月後の目標 |
|------------|-----------|-----------|
| 400点台 | 500から550点 | 600から650点 |
| 500点台 | 600から650点 | 700から750点 |
| 600点台 | 700から750点 | 800点前後 |
| 700点台 | 780から830点 | 850から900点 |

## 社会人が英語学習を続けるコツ

挫折する人が多いのは「続けられない」から。続けるためのコツを調べた。

### 習慣化のテクニック

| テクニック | 具体例 |
|----------|--------|
| 時間を固定する | 「毎朝6:30から7:00は英語」と決める |
| 既存の習慣にくっつける | 通勤電車に乗ったら単語帳を開く |
| ハードルを下げる | 「1日5分だけ」から始める |
| 記録する | アプリの連続学習日数を途切れさせない |
| 仲間を作る | オンラインの学習コミュニティに参加 |

### やってはいけないこと

| NG行動 | 理由 |
|--------|------|
| 最初から毎日2時間の計画を立てる | 挫折する。まず30分から |
| 教材を何冊も買う | 1冊を完璧にするほうが効果的 |
| 映画やドラマだけで学ぶ | 楽しいが、体系的な学習にならない |
| スコアが上がらないとすぐ方法を変える | 3ヶ月は同じ方法を続けないと効果が見えない |

## TOEIC以外の英語資格

| 資格 | 特徴 | 転職での評価 |
|------|------|-----------|
| TOEIC L&R | リスニングとリーディング。最もメジャー | 非常に高い |
| TOEIC S&W | スピーキングとライティング | 外資系で高評価 |
| 英検準1級 | 4技能バランス型 | 教育業界で高評価 |
| IELTS | 海外留学・移住向け | 外資系で高評価 |
| TOEFL | 北米の大学留学向け | 留学経験の証明 |

転職市場での汎用性はTOEIC L&Rが圧倒的。まずはここを800点に持っていくのが最も効率的。

## まとめ

コスパ最強は「アプリ+参考書+オンライン英会話」の組み合わせ。月9,000円で始められて、半年あればTOEICスコアは100から200点上がる。

自分もまずはこの組み合わせで半年やってみることにした。目標はTOEIC 800点。達成したら、その過程と結果もまた記事にする。
